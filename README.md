# 《我的中餐厅》v0.1 内测版

> 微信小游戏 - 餐厅经营模拟

## 🎮 项目简介

**我的中餐厅** 是一款轻松休闲的餐厅经营模拟小游戏。玩家扮演餐厅老板，通过接待顾客、管理厨师、升级菜品、装修餐厅等方式，打造属于自己的中餐厅帝国。

### 版本信息
- **版本**: v0.1 内测版
- **平台**: 微信小游戏
- **范围**: 内部测试（10-20 人体验）
- **目标**: 快速验证核心玩法

---

## ✨ 核心功能

### 🏠 餐厅经营
- 招揽顾客，赚取金币
- 收取收益，扩大经营
- 提升餐厅等级和声誉

### 🎨 装修系统
- 拖拽摆放家具
- 自定义餐厅布局
- 多种家具可选

### 👨‍🍳 厨师管理
- 雇佣新厨师
- 升级厨师技能
- 提升烹饪效率

### 🍜 菜品管理
- 解锁新菜品
- 升级菜品等级
- 优化收益结构

## 🚀 快速开始

### 查看原型
直接在浏览器中打开 `src/prototype.html` 即可体验交互原型。

```bash
# Mac
open src/prototype.html

# Windows
start src/prototype.html

# Linux
xdg-open src/prototype.html
```

### 微信小游戏开发
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目目录
3. 选择小游戏项目
4. 开始开发
