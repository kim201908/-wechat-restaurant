## 1. 游戏核心循环设计

### 1.1 三层循环架构

```
┌─────────────────────────────────────────────────────────┐
│                    长期循环（数周）                       │
│   餐厅等级 1→50 → 解锁新区域 → 新玩法 → 新目标            │
└─────────────────────────────────────────────────────────┘
                         ↑ ↓
┌─────────────────────────────────────────────────────────┐
│                    中期循环（每天）                       │
│   每日任务 → 达成目标 → 获得奖励 → 解锁新内容              │
└─────────────────────────────────────────────────────────┘
                         ↑ ↓
┌─────────────────────────────────────────────────────────┐
│                    短期循环（单局）                       │
│   顾客进店 → 点餐 → 制作 → 上菜 → 收金币 → 升级          │
└─────────────────────────────────────────────────────────┘
```

### 1.2 短期循环（单局 3-5 分钟）

| 环节 | 时长 | 玩家行为 | 乐趣点 | 反馈 |
|-----|------|---------|-------|------|
| 顾客进店 | 即时 | 观察顾客类型 | 收集欲、期待感 | 顾客气泡提示 |
| 点餐 | 5-10s | 点击顾客→选择菜品 | 决策感、匹配感 | 订单确认动画 |
| 制作 | 2-8s | 点击厨师/设备 | 操作感、节奏感 | 进度条 + 音效 |
| 上菜 | 2-3s | 拖拽菜品到顾客 | 成就感、流畅感 | 顾客满意表情 |
| 收金币 | 即时 | 点击金币/顾客 | 收获快感 | 金币飞入动画 + 音效 |
| 升级决策 | 30-60s | 选择升级项 | 成长感、规划感 | 升级动画 + 属性提升 |

**核心乐趣公式**：
```
单局乐趣 = 操作流畅度 × 反馈及时性 × 成长可见度
```

### 1.3 中期循环（每天 10-15 分钟）

| 时间段 | 玩家行为 | 目标 | 奖励 |
|-------|---------|------|------|
| 登录 | 领取登录奖励 | 连续登录 | 金币 + 钻石 |
| 日常经营 | 完成每日任务 | 5-8 个任务 | 金币 + 经验 + 道具 |
| 离线收益 | 收取离线金币 | 离线时长奖励 | 离线金币（上限 8 小时） |
| 限时活动 | 参与时段活动 | 活动任务 | 限定装饰/员工 |

**每日登录动机**：
1. **沉没成本**：连续登录奖励递增（第 7 天大奖）
2. **损失厌恶**：每日任务刷新，不做就没了
3. **成长焦虑**：离线收益有上限，不领就亏了
4. **社交压力**：好友排行榜，不想落后

### 1.4 长期循环（数周 - 数月）

| 周期 | 目标 | 解锁内容 | 新玩法 |
|-----|------|---------|-------|
| 第 1 周 (1-10 级) | 熟悉核心玩法 | 基础菜品、基础装饰 | 无 |
| 第 2 周 (11-20 级) | 深度经营 | 新菜品、员工系统 | 员工培训 |
| 第 3 周 (21-30 级) | 扩张 | 二楼包厢、外卖系统 | 外卖订单 |
| 第 4 周 (31-40 级) | 品牌化 | 连锁分店、特殊顾客 | 分店管理 |
| 第 5 周 + (41-50 级) | 巅峰 | 顶级装饰、美食家顾客 | 美食评论家挑战 |

**长期追求**：
- **数值追求**：餐厅等级 50 级满级
- **收集追求**：全菜品解锁、全装饰收集、全员工收集
- **排名追求**：好友榜第一、全区前 100
- **成就追求**：全成就解锁（50+ 个成就）

## 2. 吸引力设计（前 3 分钟）

### 2.1 前 30 秒 - 第一印象

| 时间点 | 内容 | 设计目标 |
|-------|------|---------|
| 0-3s | 启动画面：中国风招牌 + 欢快 BGM | 建立中国风认知 |
| 3-5s | 加载进度条 + 美食图片轮播 | 降低等待焦虑 |
| 5-10s | 新手引导 1：点击第一个顾客 | 零门槛交互 |
| 10-20s | 新手引导 2：制作第一道菜 | 核心玩法教学 |
| 20-30s | 新手引导 3：收取第一笔金币 | **第一个正反馈** |

**第一眼看到什么**：
- 温馨的中餐厅场景（暖色调、中国风装饰）
- 1-2 个可爱顾客在等待（气泡显示"好饿"）
- 明显的引导箭头指向可交互元素
- 顶部显示：金币 0、餐厅等级 1、今日营业额 0

**第一个交互**：
```
引导文字："点击顾客接订单！"
↓
玩家点击顾客
↓
弹出简单订单界面（只有 1 个菜品可选）
↓
引导文字："点击厨师开始制作！"
```

**第一个正反馈**：
```
顾客满意离开
↓
金币 +15 动画（大字体 + 金色闪光 + 音效）
↓
弹出提示："太棒了！赚取第一桶金！"
↓
自动解锁：新菜品【水饺】
```

### 2.2 前 3 分钟 - 完成第一个小闭环

| 时间 | 事件 | 成就 | 奖励 |
|-----|------|------|------|
| 30s-1min | 服务 3 个顾客 | "开门大吉"成就 | 金币 +50 |
| 1-2min | 升级第一个设施（炉灶 Lv.2） | "精益求精"成就 | 解锁【炒饭】 |
| 2-3min | 餐厅等级提升到 2 级 | "初出茅庐"成就 | 解锁【装饰系统】 |

**3 分钟结束时玩家状态**：
- 金币：约 200-300
- 餐厅等级：2 级
- 解锁菜品：4 个（拉面、水饺、炒饭 + 新解锁 1 个）
- 解锁功能：基础经营 + 设施升级 + 装饰系统
- 心理状态："有点意思，知道怎么玩了，想继续试试"

### 2.3 前 10 分钟 - 形成期待

| 时间 | 挑战 | 决策 | 期待形成 |
|-----|------|------|---------|
| 3-5min | 顾客变多，手忙脚乱 | 先升级厨师还是先买新桌椅？ | "我要优化效率" |
| 5-7min | 金币不够升级所有设施 | 优先升级哪个？ | "我要做战略规划" |
| 7-10min | 遇到 VIP 顾客（小费高） | 如何服务得更好？ | "我要赚更多" |

**10 分钟时的期待**：
- "升到 5 级能解锁什么？"
- "听说后面有外卖系统，什么时候开？"
- "好友排行榜我能排第几？"
- "明天登录能拿什么奖励？"

### 2.4 前 3 分钟流程图

```
[启动游戏]
    ↓
[加载画面 5s] → 美食图片轮播 + 进度条
    ↓
[进入餐厅场景] → 温馨中国风 + 2 个等待顾客
    ↓
[引导 1: 点击顾客] → 弹出订单界面
    ↓
[引导 2: 点击厨师] → 开始制作（进度条 3s）
    ↓
[引导 3: 拖拽上菜] → 顾客满意表情
    ↓
[金币 +15 动画] → ⭐ 第一个正反馈 ⭐
    ↓
[自动解锁：水饺] → 新菜品提示
    ↓
[服务满 3 个顾客] → 成就"开门大吉" + 金币 +50
    ↓
[引导 4: 升级炉灶] → 炉灶 Lv.1→Lv.2
    ↓
[餐厅经验满] → 等级 1→2
    ↓
[解锁装饰系统] → 引导购买第一个装饰
    ↓
[3 分钟结束] → 玩家状态：金币 200+、等级 2、菜品 4 个
```

### 4.2 员工成长线

#### 服务员（5 阶×10 级 = 50 级）

| 阶位 | 等级范围 | 名称 | 基础能力 | 解锁条件 |
|-----|---------|------|---------|---------|
| 1 阶 | 1-10 级 | 新手服务员 | 移动速度 1.0× | 初始解锁 |
| 2 阶 | 11-20 级 | 熟练服务员 | 移动速度 1.3× | 餐厅 Lv.5 |
| 3 阶 | 21-30 级 | 资深服务员 | 移动速度 1.6× + 小费 +10% | 餐厅 Lv.15 |
| 4 阶 | 31-40 级 | 明星服务员 | 移动速度 2.0× + 小费 +20% | 餐厅 Lv.30 |
| 5 阶 | 41-50 级 | 金牌服务员 | 移动速度 2.5× + 小费 +30% + VIP 服务 | 餐厅 Lv.45 |

**服务员升级**：
- 每级消耗：金币 50-500 + 经验药水
- 经验来源：服务顾客（1 顾客=10 经验）

## 9. 竞品分析与差异化

### 9.1 竞品分析

#### 《动物餐厅》

| 维度 | 动物餐厅 | 我们的策略 |
|-----|---------|-----------|
| 核心玩法 | 放置 + 收集 | 放置 + 经营 + 社交 |
| 美术风格 | 治愈系动物 | 中国风人物 |
| 付费点 | 小鱼干（加速） | 钻石（多元付费） |
| 社交 | 弱社交 | 强社交（拜访、偷取） |
| 内容量 | 大量菜品、设施 | 相当 + 中国菜系深度 |
| 更新频率 | 月度大更新 | 双周活动 + 月度更新 |

**借鉴点**：
- ✅ 放置收益机制
- ✅ 收集驱动（菜谱、设施）
- ✅ 治愈系体验

**差异化**：
- 🎯 中国风主题（文化认同）
- 🎯 更强社交互动
- 🎯 更丰富的员工系统
- 🎯 外卖/分店等扩展玩法

#### 《楼下的早餐店》

| 维度 | 早餐店 | 我们的策略 |
|-----|-------|-----------|
| 核心玩法 | 时间管理 + 经营 | 放置 + 时间管理混合 |
| 节奏 | 快节奏操作 | 可快可慢（放置为主） |
| 关卡 | 关卡制 | 无限经营 + 等级制 |
| 故事 | 有剧情 | 轻剧情（顾客故事） |
| 难度 | 较高（手速要求） | 较低（休闲友好） |

**借鉴点**：
- ✅ 时间管理乐趣
- ✅ 顾客故事系统
- ✅ 关卡目标感

**差异化**：
- 🎯 无限经营（无压力）
- 🎯 放置收益（离线也能赚）
- 🎯 社交互动（好友系统）
- 🎯 深度养成（员工、设施）

**借鉴点**：

**差异化**：
