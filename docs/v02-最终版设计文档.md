## 2. 核心功能

### 2.1 餐厅经营（自动模式）

**核心机制：**
- 顾客自动进店（无需手动操作）
- 顾客停留时间根据时段动态调整
- 离开时自动产生收益
- 收益自动计入金币池

**影响因素：**
| 因素 | 影响 | 公式 |
|-----|------|------|
| 餐厅等级 | 客流密度 | 基础间隔 = 10 秒 / (等级 × 0.5) |
| 时段 | 客流倍率 | 早餐×1.0、午市×1.5、晚市×2.0、深夜×0.5 |
| 空桌数量 | 来客条件 | 有空位才来顾客 |
| 餐厅评分 | 吸引力 | 评分/100（上限 2 倍） |

**顾客流程：**
```
进店 → 找座位 → 等待 → 就餐 → 结账 → 离开
(自动)  (自动)  (3-5 秒) (2-5 秒) (自动) (自动)
```

### 2.4 外卖系统

**核心流程：**
```
生成订单 → 配送中（2-4 分钟）→ 可收割 → 超时取消（2 分钟）
   ↓           ↓              ↓           ↓
 点击生成   倒计时显示    点击收割    连续归零
```

**订单机制：**
| 属性 | 说明 |
|-----|------|
| 订单金额 | ¥15-(15+ 等级×15) 随机 |
| 配送时间 | 120-240 秒（2-4 分钟） |
| 超时期限 | 可收割后 120 秒 |
| 每日上限 | 20 单/天 |

**勋章系统：**
| 勋章 | 条件 | 奖励 |
|-----|------|------|
| 🏆 连续 5 单 | 连续收割 5 单 | 成就解锁 |
| 🥇 连续 10 单 | 连续收割 10 单 | 成就解锁 |
| 👑 连续 20 单 | 连续收割 20 单 | 成就解锁 |

**每日重置：**
- 22:00 自动进入第 2 天
- 今日订单数、今日收入清零
- 连续收割归零

## 4. 游戏机制

### 4.1 时间系统

**时间速度：** 游戏时间 = 真实时间 × 60 倍

**营业时间：** 08:00 - 22:00（22:00 后进入第 2 天）

**时段定义：**
| 时段 | 时间 | 客流倍率 | 订单倍率 |
|-----|------|---------|---------|
| 早餐 | 08:00-11:00 | ×1.0 | ×1.0 |
| 午市 | 11:00-14:00 | ×1.5 | ×1.5 |
| 晚市 | 17:00-21:00 | ×2.0 | ×2.0 |
| 深夜 | 21:00-22:00 | ×0.5 | ×0.5 |

### 4.2 升级机制

**餐厅升级：**
```
经验值 = 金币收益 / 10
升级所需经验 = 等级 × 100
升级奖励：桌子数量 +1
```

**等级上限：** Lv.50

### 4.3 收益计算

**堂食收益：**
```
基础收益 = 10-20 随机
时段加成 = 基础 × 时段倍率
最终收益 = 时段加成（取整）
```

**外卖收益：**
```
基础金额 = 15-(15+ 等级×15) 随机
最终收益 = 基础金额（全额计入金币池）
```
