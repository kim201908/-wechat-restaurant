# 微信小游戏测试指南

## 📥 下载最新代码

1. 访问：https://github.com/kim201908/-wechat-restaurant
2. 点击 "Code" → "Download ZIP"
3. 解压到本地

## 🛠️ 微信开发者工具配置

1. 打开微信开发者工具
2. 点击 "+" 添加项目
3. 项目目录：选择 `wechat-minigame` 文件夹
4. AppID：`wxc5e09da0eacdcfb9`（测试号）
5. 点击"编译"

## ✅ 功能测试清单

### 1️⃣ 首页（🏠）
- [ ] 顶部状态栏显示（金币/声誉/钻石/时间/天数/厨师）
- [ ] 餐厅场景占位图
- [ ] 4 个统计卡片（金币/声誉/厨师/菜品）
- [ ] "💰 收取收益" 按钮 → 点击后金币增加
- [ ] "📈 升级管理" 按钮 → 点击切换到经营 Tab

### 2️⃣ 经营 Tab（📈）

#### 餐厅子 Tab（🏪）
- [ ] 子导航切换（餐厅/厨房/外卖）
- [ ] 装修预览区域
- [ ] 家具商店（餐桌/椅子/绿植/装饰画）

#### 厨房子 Tab（🍳）
- [ ] 厨师列表显示（名字/等级/速度）
- [ ] "📢 雇佣新厨师" 按钮 → 扣除 500 金币，新增厨师
- [ ] 金币不足提示
- [ ] 菜品列表显示（名字/等级/价格）
- [ ] 点击菜品 → 升级（扣除金币，等级 +1，价格 +5）

#### 外卖子 Tab（🛵）
- [ ] 统计卡片（今日单数/今日收入/历史收入）
- [ ] "➕ 生成订单" 按钮 → 显示加载动画

### 3️⃣ 社交 Tab（👥）

#### 好友子 Tab（👥）
