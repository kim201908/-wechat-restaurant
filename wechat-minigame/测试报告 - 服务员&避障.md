## 【代码审查发现】

### ✅ 优点

1. **状态机设计清晰** - 顾客和服务员都有明确的状态流转
2. **避障算法合理** - 使用垂直绕行策略，简单有效
3. **动画效果完整** - 包含走路颠簸效果 (`Math.sin(walkFrame)`)
4. **代码已提交** - Git 状态 clean，已推送到 GitHub

### ⚠️ 潜在问题

1. **绕行逻辑简化** - 只尝试左右两个方向，可能被多个家具围住
   ```javascript
   // main.js:365-375
   if (!checkFurnitureCollision(leftX, leftY, 15)) {
     entity.x = leftX;
     entity.y = leftY;
   } else if (!checkFurnitureCollision(rightX, rightY, 15)) {
     entity.x = rightX;
     entity.y = rightY;
   } else {
     // 都被挡住，尝试小幅度后退
   }
   ```

2. **碰撞半径固定** - 所有家具视为相同大小 (40x40)，实际家具大小可能不同

3. **性能考虑** - 每 50ms 更新所有顾客位置，大量顾客时可能影响性能
